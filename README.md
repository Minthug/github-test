# My Todo App 

---
Todo 관리를 위한 앱 애플리케이션 입니다.

## Features

---
- 편리한 UI로 Todo를 쉽게 생성하고 삭제할 수 있다.
- Todo에 기한과 카테고리를 설정할 수 있다.
- create-react-app으로 간편한 번들링과 배포가 가능하다
- Spring boot 으로 쉽게 서버를 배포할 수 있다.
